# Go Task Manager

A task management application built with Go.

## Getting Started

### Run the project

To run the project either run it directly with go:

```go run main.go```

Or use the makefile:

```make run```

### Build the project

To build the project, you can use:

```go build main.go```

Or use the makefile:

```make build```

To build and run the binary, use:

```make run-build```
